Hotel Review Sentiment

Predict sentiment for hotel review


DESCRIPTION

This sample is using TensorFlow.js to predict hotel review sentiment while user types text. Prediction is executed on the client side, without server call. This allows to provide immediate feedback to the user, this could help user to adjust hotel review before review is submitted.

EXAMPLES:

1. Positive: Location near Trafalgar square is excellent. Rooms a reasonable size for inner London
2. Negative: Strange smell in the room due to newly painted or new carpet
